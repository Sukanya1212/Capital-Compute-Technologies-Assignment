# QuickCast

A minimal Loom-like screen recorder built with React (Vite) and Tailwind CSS.

## Features

- Start/stop screen recording using browser APIs
- Optional system/tab audio capture (browser-dependent)
- Live recording timer
- In-app preview after recording
- Download recording as `.webm`
- Record again flow with clear UI states

## Tech Stack

- React + Vite
- Tailwind CSS
- MediaRecorder API + `getDisplayMedia`

## Run Locally

```bash
npm install
npm run dev
```
